Desarrollo Basado en Plataformas

  1. El curso de Desarrollo Basado en Plataformas vale 2 crétidos

Propósito del curso

El curso tiene como objetivo que los alumnos logren hablidades de diseño, implementación, y evaluación de diversos sistemas, procesos o un determinado programa computacional, para alcanzar los requirimientos, y necesediades adecuadas y deseadas. El curso de adecua al cambio provocado por el uso de la web y tecnologías relacionadas, como las aplicaciones web y las aplicaciones móviles. Es por ello que el alumno va usar herramientas y lenguajes como HTML, CSS, JavaScript(incluyendo AJAX), back-end scripting y base de datos.
