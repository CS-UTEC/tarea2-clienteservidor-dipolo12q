Resumen de la carrera

Ciencias de la computación es una carrera orientada al software, es por ello que dentro de este curso, los alumnos adquieren conocimientos de programación, redes, seguridad informáticas, base de datos, entre más cosas de este área, el cual es muy amplio. Asimismo, los estudiantes desarrollan capacidades para proponer soluciones creativas a diversos problemas en cualquier tipo de área (educación, salud, negocios, entre otros), esto debido a que el software esta vinculado a todo.
